# starmusic

#### 介绍
vue+springboot+mybatisPlus写的一个小小的网页播放音乐 望大家多多指正 小白初学

#### 软件架构
软件架构说明
后端 java语言 springboot
前端 vue
数据库 mysql
jdk 17
